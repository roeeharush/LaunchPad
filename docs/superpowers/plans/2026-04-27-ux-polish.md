# UX Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Sonner toasts, Tailwind skeleton loaders, a nextjs-toploader progress bar, and individual save-button spinners to the Launchpad dashboard without touching any server actions.

**Architecture:** Two new UI primitive files (`skeleton.tsx`, `skeleton-card.tsx`) are created. The protected layout receives two singleton mounts (`NextTopLoader`, `Toaster`). Six existing client components are updated in-place to fire toasts and swap skeleton placeholders during loading states. All changes are purely additive — no server actions or data-fetching logic is modified.

**Tech Stack:** sonner ^2, nextjs-toploader ^3, Tailwind v4 `animate-pulse`, React 19 `useTransition`, vitest 4, @testing-library/react 16
